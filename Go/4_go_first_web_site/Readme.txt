// =========================================


Run go file (without build):
	
	$ go run main.go

Build executable file:

	$ go build main.go

Run build file in terminal:

	$ ./main

// =========================================

Add import from github

	$ go mod init modulename

Add dependencies:
	
	// mysql
	$ go get -u github.com/go-sql-driver/mysql

	// router
	$ go get -u github.com/gorilla/mux
