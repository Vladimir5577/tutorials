1. In the first place install composer in any directory
2. Then run command 
	
	$ mv composer.phar /usr/local/bin/composer

3. Then check 

	$ composer

// ======================
Install symfony globally

	$  wget https://get.symfony.com/cli/installer -O - | bash
	It will create in home directory ~/.symfony/bin/symfony
	Copy symfony file to 
	$ sudo cp symfony /user/local/bin 
	Then check 
	$ symfony
