## Create package json

```bash
$ npm init -y
```

## Install express

```bash
$ npm install express
```
## Inside project create 'src' folder.
## In 'src' folder main file 'app.js'.

## Install nodemon

```bash
$ npm install nodemon
```
## In 'package.json' file write start path

```bash
"scripts": {
	...
	"start": "nodemon src/app.js"
	...
}
```

## Run application 
```bash
$ npm start
```
## To work with files install handlebar
```bash
$ npm install hbs
```
