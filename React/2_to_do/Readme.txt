Create React

	$ npx create-react-app my-app
	$ cd my-app
	$ npm start // yarn start

Go to src/App.js and remove <div className="App"> completely, or just header.
	Put <h1>Hello World</h1> to see how it work.
	Change function on class
	Remove import logo from './logo.svg';
	Remove files:
		- src/App.test.js
		- index.css
		- logo.svg

To add animation 

	$ npm i -S react-flip-move


Build:

	$ npm run build
